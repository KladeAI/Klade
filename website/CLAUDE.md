# CLAUDE.md — Klade Website

## What this is
The kladeai.com marketing website. Next.js 16 app with Tailwind CSS, Framer Motion, deployed on Vercel.

## Stack
- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Language:** TypeScript
- **Deployment:** Vercel
- **Package manager:** npm

## Commands
```bash
npm run dev    # local dev server
npm run build  # production build
npm run lint   # eslint
```

## Working rules
- Follow the parent workspace CLAUDE.md for all operating rules
- This is a marketing site — speed, clarity, and polish matter
- Keep it lightweight — no unnecessary dependencies
- Supabase integration exists under `supabase/` dir
- Design prompt and V3 task spec are in `WEBSITE_PROMPT.md` and `WEBSITE_V3_TASK.md`
